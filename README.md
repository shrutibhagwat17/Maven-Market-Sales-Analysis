# Maven Market Sales Analysis Dashboard

## Project Overview
An interactive Power BI dashboard designed to analyze sales performance, profitability, transactions, and return rates across brands and regions.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Metrics
- Total Transactions
- Total Profit
- Profit Margin
- Return Rate

## Features
- KPI Tracking
- Geographic Analysis
- Weekly Revenue Trends
- Interactive Filters
- Conditional Formatting

## Key Insights
- Transactions exceeded target by 28%.
- Profit exceeded target by 30%.
- Return rate remained above target and requires monitoring.
- Regional and brand-level performance analysis helps identify growth opportunities.
